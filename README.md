# Data Science for Public Policy 2026: Public Handout Repo

Public handout subset for the course `Data Science for Public Policy 2026`.

This repository is a sanitized public version of the main teaching repository. It includes only materials that are currently suitable for open distribution.

## Included

- Lecture 1 PDF slides
- Lecture 2 PDF slides
- Lecture 2 Homework 1 instructions
- Lecture 2 synthetic practice dataset

## Not Included

- Private course management files
- Raw survey response files
- Internal teaching drafts and source files
- Materials containing class-specific calibration results
